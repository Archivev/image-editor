## image-editor

这是一个`React`截图工具的组件, 基本功能和微信截图相似, 微信截图是一个相对比较成熟且稳定的产品, 所以我们的功能目前在一定程度上比不上微信截图, 但基本满足截图的所有场景, 因为使用了`React Hooks`, 所以对`React`的版本有一定要求

## install

```shell
yarn add -S image-editor-little
```
or
```shell
npm install -S image-editor-little
```

<ImageEditor
